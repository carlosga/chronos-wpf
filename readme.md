# CHRONOS WPF

**Old  project initally hosted at Codeplex.**

Chronos is a framework to build Multiple Document Interfaces (MDI) applications using Windows Presentation Foundation. 

# Dependencies

* Microsoft Visual Studio 2010 or Microsoft Expression Blend 4.0. 
* Microsoft .NET 4.0 
* [nRoute](http://nroute.codeplex.com) 
* [Microsoft Expression Blend SDK 4](http://www.microsoft.com/downloads/en/details.aspx?FamilyID=d197f51a-de07-4edf-9cba-1f1b4a22110d&displaylang=en). 
* [NLog](http://nlog-project.org/)

# Credits 

## nRoute. 

nRoute is a composite application framework for creating MVVM-style applications in Silverlight, WPF, and Windows Phone 7, 
more information on nRoute can be found at the project site in [codeplex](http://nroute.codeplex.com/)  and the nRoute author [blog](http://www.orktane.com/Blog/). 

## Silverlight Cosmopolitan Theme The Chronos Metro Theme

Ported to Windows Presentation Foundation from the [Cosmopolitan Silverlight theme](http://timheuer.com/blog/archive/2010/05/03/new-silverlight-4-themes-available-for-download.aspx). 

## Extension Methods library

Most of the extensions methods available in Chronos.Extensions and Chronos.Extensions.Windows Visual Studio projects classes 
are based on the [.NET Extensions - Extension Methods Library for C# and VB.NET](http://dnpextensions.codeplex.com/). 

## Drag & Drop support 

Based on the Jaime Rodriguez blog posts & samples, about the Windows Presentation Foundation drag & drop support. 

## MoonyDesk

The digital clock widget is based on one of the [MoonyDesk](http://moonydesk.codeplex.com/) widgets. 

## Watermark Textbox Behavior 

The Watermark Textbox Behavior is based on this [blog post](http://blindmeis.wordpress.com/2010/07/16/wpf-watermark-textbox-behavior/).
